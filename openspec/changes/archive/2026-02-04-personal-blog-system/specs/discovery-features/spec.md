## 新增需求

### 需求：站内搜索
系统必须提供全文搜索功能，允许用户通过关键词查找文章标题和内容。搜索响应必须迅速（建议 < 100ms）。

#### 场景：执行搜索
- **当** 用户在搜索框输入关键词
- **那么** 实时/快速显示匹配的文章列表，并高亮匹配词

### 需求：标签系统
系统必须支持标签归档，允许用户点击标签查看所有相关文章。

#### 场景：按标签筛选
- **当** 用户点击某个标签（例如 "CSS"）
- **那么** 页面显示所有带有 "CSS" 标签的文章列表

### 需求：RSS 订阅
系统必须自动生成 RSS 2.0 或 Atom 订阅源文件。

#### 场景：RSS 抓取
- **当** RSS 阅读器访问 `/rss.xml`
- **那么** 系统返回包含最新文章内容的有效 XML 文件
