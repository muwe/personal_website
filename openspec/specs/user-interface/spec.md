## 新增需求

### 需求：首页文章列表
首页必须以极简列表形式展示最新的文章，包含标题、发布日期和标签。列表应按日期倒序排列。

#### 场景：查看首页
- **当** 用户访问网站根路径
- **那么** 页面显示最新的文章列表，每项包含标题和格式化的日期

### 需求：文章详情页
文章详情页必须专注于阅读体验，移除无关干扰。必须包含文章标题、元信息、正文内容、版权声明（可选）和标签链接。

#### 场景：阅读文章
- **当** 用户点击文章标题
- **那么** 跳转至详情页，正文宽度被限制在最佳阅读宽度（例如 65ch），排版舒适

### 需求：导航系统
系统必须提供清晰的全局导航，包括“首页”、“归档/标签”、“关于”等链接。

#### 场景：页面跳转
- **当** 用户点击导航栏中的链接
- **那么** 页面应无刷新跳转（View Transitions）至目标页面
