## 新增需求

### 需求：文章内容处理
系统必须支持解析 Markdown (`.md`) 和 MDX (`.mdx`) 文件作为文章源。系统必须验证 Frontmatter 元数据（标题、日期、标签）。

#### 场景：渲染文章内容
- **当** 系统构建文章页面
- **那么** Markdown 内容应被转换为 HTML，并在构建时进行验证

### 需求：代码高亮
系统必须支持代码块的语法高亮，显示行号，并支持复制按钮。

#### 场景：显示代码块
- **当** 文章中包含代码块（例如 ```js ... ```）
- **那么** 代码块应带有语法颜色高亮，背景色符合当前主题

### 需求：图片优化
系统必须自动优化文章内引用的图片，生成响应式尺寸和 WebP 格式。

#### 场景：加载文章大图
- **当** 用户浏览包含大图的文章
- **那么** 浏览器应加载适配当前屏幕的优化后图片版本，防止布局偏移（CLS）
